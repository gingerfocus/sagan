* Problem 2
Count the exact number of “sum++” operations (as a function of N) of each of following code fragments.
** Part A

#+begin_src C
int sum = 0;
for (int i = 1; i < N; i++)
    for (int j = 0; j < i; j++)
        sum++;
#+end_src

Answer: \(1 + 2 + 3 + 4 + ... + (n-2) + (n-1) = \frac{(n-2)(n-1)}{2}\)

** Part B
#+begin_src C
int sum = 0;
for (int i = 1; i < N; i *= 2)
    for (int j = 0; j < N; j++)
        sum++;
#+end_src

Answer: \(1 + 2 + 4 + ... + (\frac {n}{2}) + (n) = \frac{1-2^(n+1)}{1-2} = 2^{n+1} - 1\)
