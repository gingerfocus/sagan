* Problem 1
/Given an array A, an inversion in A is defined as a pair of indices i and j such that i < j but A[i] > A[j]. Design a O(n log n) running time algorithm for computing the number of inversions in an input array A./

#+begin_quote
[!tip] Algorithm 1) When the list has only 1 element return 0. 2) Split the list in the middle and recurse on both halves, adding count of each to sum. 3) Merge lists together in ascending order. When left side is greater, add amount items remaining on right side to sum. 4) Return sum.
#+end_quote

#+begin_quote
[!example] Example
An example implementation using zig language.

#+begin_src zig
fn countInversions(L: []u8) !usize {
   if (L.len <= 1) return 0;

   // allocate buffer for merge sorted list
   const a = std.heap.page_allocator;
   var buf = try a.alloc(u8, L.len);
   defer a.free(buf);

   var sum: usize = 0;
   const half = L.len / 2;
   sum += try countInversions(L[0..half]);
   sum += try countInversions(L[half..]);

   var lhs: usize = 0;
   var rhs: usize = half;
   var ptr: usize = 0;
   while (lhs < half or rhs < L.len) { // while either side has more
       // if there is nothing on rhs or lhs is strictly greater
       if (rhs == L.len or L[lhs] > L[rhs]) {
           buf[ptr] = L[lhs];
           sum += L.len - rhs; // add number of elements it is greater than
           lhs += 1;
       } else { // if right side is greater OR EQUAL
           buf[ptr] = L[rhs];
           rhs += 1;
       }
       ptr += 1;
   }

   // copy sorted list to input
   for (buf, 0..) |v, i| L[i] = v;

   return sum;
}
#+end_src
#+end_quote

* Problem 3
/Consider the one-dimensional version of the closest-pair problem in which you are given as input a set of n points, P, all of which lie on the x-axis. The output is the Euclidean distance between the closest pair of points in P. Design and analyze a divide and conquer algorithm for this problem./

#+begin_quote
[!tip] Algorithm 1) Sort the list with merge sort in first iteration, assume the list is sorted based on x-values in others. 2) Recurse on each half of the list, record the minimum result. 3) Compute the distance of the last element in the first split to the first element in the second split. Update minimum if less. 4) Return the minimum value.
#+end_quote
