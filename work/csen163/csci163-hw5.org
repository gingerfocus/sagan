#+begin_quote
[!faq] Question
You are given an array of n real numbers and another real number s. The goal is to find out whether the array contains two elements whose sum is s. Design and analyze a transform-conquer algorithm for this problem.
#+end_quote

#+begin_quote
[!tip] Algorithm 1) Sort the list using merge sort. 2) Iterate the list. 3) For each item: use binary search to find the complement (s - v) and return true if found. 4) If no item exits return false.
#+end_quote

#+begin_quote
[!qna] Analysis 1) \(T(n) = O(n \log n) + n O(\log n) = O(n \log n) + O(n \log n)\) 2) \(T(n) = 2O(n \log n) = O(2n \log n)\) 3) \(T(n) = O(n \log n)\)
#+end_quote
