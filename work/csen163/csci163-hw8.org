* Problem 1

#+begin_quote
[!faq] Problem
Given a digraph G = (V, E), its transitive closure G* is a digraph defined as follows: It has the same vertex set V as G and contains an edge from a vertex a to another vertex b if and only if there is a directed path from a to b in G. Suppose we want to design a DP algorithm that outputs G∗ on input G.
#+end_quote

** Solution
Consider a 2x2 adjacency matrix where M[a][b] means the edge from a to b is in G*. Initially, fill the list with the the edges of G and the diagonal M[a][a] as each edge can reach itself. Then you can iterate each set of 3 vertices. If there is a directed path from a to b and b to c then there is a directed path from a to c. After iterating the set of vertices 3 times G* is represented by the adjacency matrix so the runtime is \(O(V^3+E)\).
