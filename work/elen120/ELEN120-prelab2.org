* Calculations
In the lab, we will write a program to square and sum elements in an array. To
aid the lab, these values can be compared to a calculated value.

The array is defined as:
#+BEGIN_SRC asm
array DCD 1, -5, 2, 1, 1, 0, 10, 0, -2, -1
#+END_SRC

Thus its sum ($S$) is:
$ S = 1^2 + (-5)^2 + 2^2 + 1^1 + 1^1 \
  + 0^2 + 10^2 + 0^2 + (-2)^2 + (-1)^2 $
$ S = 1 + 25 + 4 + 1 + 1 + 0 + 100 + 0 + 4 + 1 $
$ S = 30 + 2 + 105 = 137 $
